\CWHeader{Лабораторная работа №7}

\CWProblem{
    При помощи метода динамического программирования разработать алгоритм решения задачи; оценить время выполнения алгоритма и объем затрачиваемой оперативной памяти. \\
{\bfseries Вариант задачи: } Задано целое число n. Необходимо найти количество натуральных (без нуля) чисел, которые меньше n по значению и меньше n лексикографически (если сравнивать два числа как строки), а так же делятся на m без остатка.
}
